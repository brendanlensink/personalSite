Hey look, it's a Minecraft server!

How to Connect

  1. You'll need a copy of Minecraft: Java edition. You can purchase and download it here.
  2. Inside Minecraft, click Multiplayer > Add Server, then in the Server Address field, enter 142.4.207.225:25657, and hit done.
  3. Now select the Steamblocks server and press Join Server, and you're all set!

Some useful commands

/sethome: Sets a home point.
/home: After 5 seconds, warps you to the point you set with /sethome.
